**日期**: 2026-06-03

## 1. 产品概述

寰卓是一个基于**双重供给+三层定价**的AI API商业化平台，将阿里云ModelRouter官方API与寰卓自研算力调度网络（闲散算力池）融合，为C端消费者和B端企业提供高性价比的AI推理服务。

### 核心理念

```
同样的模型，更便宜的价格
你的GPU也能赚钱
绿色AI，零碳推理
智能调度，永远最优
```

## 2. 用户画像

### 2.1 C端消费者
- **学生**: 价格敏感，需要经济套餐，主要用于学习和创作
- **个人开发者**: 需要稳定API，对价格敏感，需要经济套餐
- **AIGC创作者**: 批量推理需求，可容忍轻微延迟，追求极致性价比
- **AI爱好者**: 体验为主，需要低门槛入门套餐

### 2.2 B端客户
- **中小企业**: 需要稳定API，对SLA有要求，选择标准套餐
- **互联网大厂**: 峰值波动明显，需要混合调度方案
- **AIGC平台**: 大量推理需求，追求成本优化
- **高校/研究机构**: 预算有限，需要经济套餐+闲时托管

### 2.3 算力提供者（GPU合伙人）
- **企业自有GPU**: 闲时段利用率低，希望通过托管赚取收益
- **IDC运营商**: 闲时算力资源，希望通过寰卓平台变现
- **网吧/个人PC**: 高端GPU，闲时加入算力网络

## 4. 核心功能

### 4.1 智能调度网关

```
用户请求 → 请求特征分析 → 客户等级匹配 → 实时比价 → 最优供给池选择
```

**调度策略**:
- 实时对话/低容错 → 标准池（ModelRouter）
- 批量任务/高容错 → 经济池（寰卓算力池）
- 企业客户 → 智能混合调度

### 4.2 GPU合伙人计划

**核心机制**: 用户可将自有/长租GPU的闲时时段接入寰卓调度池，赚取Token积分或现金分成

**收益分成**:
- 算力提供者: 60%
- 寰卓平台: 25%
- 渠道/代理: 15%

**产品功能**:
1. GPU接入向导（一键安装客户端）
2. 闲时设置（自定义闲置时段）
3. 收益看板（实时显示在线时长、贡献算力、赚取Token）
4. 安全隔离（Namespace/eBPF隔离技术）
5. 自动提现（积分抵扣套餐或满100元提现）

### 4.3 绿色AI认证

| 绿色认证 | 机制 | 营销价值 |
|----------|------|----------|
| 绿电消纳 | 任务优先调度至绿电富余区域 | 每万次调用减少X kg碳排放 |
| 闲时利用 | 利用本将被浪费的算力资源 | "让每一张GPU物尽其用" |
| 碳足迹追踪 | 为用户显示API调用的碳排放数据 | 企业ESG报告素材 |
| 绿色徽章 | 选择经济套餐获得"绿色AI"徽章 | 可用于客户产品宣传 |

## 9. 风险与应对

| 风险 | 影响 | 应对策略 |
|------|------|----------|
| 算力池稳定性不足 | 经济套餐SLA不达标 | 智能降级+熔断机制 |
| 客户对闲散算力不信任 | 转化率低 | 透明展示+安全认证 |
| 数据安全顾虑 | 企业客户拒绝 | 四级隔离+等保认证 |
| 供应商依赖 | 供应商锁定 | 双供应商+退出条款 |
| 价格竞争 | 算力池价格波动 | 动态定价+保底价 |
