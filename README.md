> **定位**: 将分散的闲散算力（IDC闲时、网吧设备、个人PC、公有云余量、政企算力集群）通过自研Rust调度系统，转化为稳定的企业级AI推理服务，帮助客户降低推理成本最高50%。

## 📦 三层产品体系

### 标准套餐 (ModelRouter供给)
- **定位**: 企业级客户、高价值应用、合规敏感场景
- **供给**: 阿里云ModelRouter官方API
- **SLA**: 99.99%
- **定价**: 体验¥19.9 / 创作¥59.9 / 专业¥199

### 经济套餐 (寰卓闲散算力池)
- **定位**: 价格敏感客户、批量推理、可容忍轻微延迟
- **供给**: 寰卓闲散算力池 (IDC闲时/网吧/云余量)
- **SLA**: 99.9% (允许秒级中断，自动重试)
- **定价**: 体验¥9.9 / 创作¥39.9 / 专业¥129 (**5-7折**)

### 企业定制 (混合调度)
- **定位**: 大客户、峰值波动明显、需要定制化服务
- **供给**: ModelRouter (基线) + 寰卓池 (弹性溢出)
- **SLA**: 99.99% (基线) + 99.5% (弹性层)
- **定价**: 按需协商，最低¥0.005/千Token起

## 🎯 快速启动

```bash
# 1. 克隆代码
git clone https://github.com/griffithjin/huazhuo.git
cd huazhuo

# 2. 配置环境变量
cp backend/.env.example backend/.env
# 编辑 .env 填入：
# - 支付宝 AppID / 公钥 / 私钥
# - 阿里云 AccessKey / Secret / 折扣率
# - 阿里云短信 AccessKey / 签名 / 模板Code

# 3. 一键启动
bash scripts/deploy.sh start

# 4. 查看状态
bash scripts/deploy.sh status
```
