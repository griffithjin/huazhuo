### 3. 服务内容与范围

#### 3.1 平台提供的服务

1. **AI Token 交易服务**：提供各大 AI 模型的 API 调用能力，以 Token 为计费单位
2. **套餐购买服务**：提供多种预付费套餐供用户选择
3. **渠道推广服务**：为认证渠道商提供推广链接和佣金结算
4. **账户管理服务**：提供余额查询、充值、消费记录等功能
5. **API 调用服务**：提供标准 REST API 接口供开发者集成

#### 3.2 服务限制

- 平台保留随时调整服务范围、功能或定价的权利
- 某些服务可能仅面向特定类型用户（如企业用户）开放
- 平台不对第三方 AI 模型的可用性、准确性或性能做出保证

### 4. 账户注册与管理

#### 4.1 注册资格

- 您必须年满 18 周岁，或根据适用法律具备完全民事行为能力
- 您必须提供真实、准确、完整的注册信息
- 每个用户仅限注册一个账户

#### 4.2 账户安全

- 您应妥善保管账户密码和 API Key，对账户下的所有活动负责
- 如发现账户被盗用或存在安全风险，应立即通知平台
- 平台有权在发现异常活动时临时冻结账户

#### 4.3 账户注销

- 您可随时申请注销账户
- 账户注销后，未消费的余额将按照退款政策处理
- 注销后，相关数据将按照隐私政策规定的期限保留

### 5. 服务费用与支付

#### 5.1 定价与计费

- 平台采用 Token 用量计费模式，不同模型有不同的单价
- 套餐价格以购买时页面显示为准
- 平台保留调整定价的权利，调整前将通过适当方式通知用户

#### 5.2 支付方式

- 平台支持支付宝、微信支付等支付方式
- 支付过程由第三方支付机构处理，平台不直接处理您的支付信息

#### 5.3 退款政策

| 情形 | 处理方式 |
|------|----------|
| 购买后未使用任何 Token | 7 天内可申请全额退款 |
| 套餐已部分使用 | 按剩余未使用比例退款，扣除已使用部分 |
| 服务故障导致无法使用 | 经核实后，按受影响时长比例退款 |
| 用户主动注销账户 | 余额部分按剩余金额退款 |

**以下情形不予退款：**
- 已过期或已完全使用的套餐
- 因用户违反本条款导致账户被封禁
- 因不可抗力导致的服务中断

#### 5.4 发票

- 用户可在支付完成后申请开具发票
- 发票类型和内容以实际交易为准
- 电子发票将在申请后 [3] 个工作日内开具

### 6. 用户行为规范

#### 6.1 禁止行为

用户在使用平台服务时，不得从事以下行为：

1. **违法违规**：使用平台服务进行任何违法、违规或侵犯第三方权益的活动
2. **欺诈行为**：提供虚假信息、恶意退款、洗钱等
3. **滥用服务**：通过技术手段绕过平台限制、批量注册账户、刷量等
4. **攻击行为**：对平台或第三方系统进行攻击、入侵、破坏
5. **内容违规**：生成、传播违法、色情、暴力、歧视性内容
6. **数据抓取**：未经授权抓取平台数据或用户信息
7. **转售行为**：未经平台授权，将平台服务转售给第三方

#### 6.2 违规处理

如发现用户存在违规行为，平台有权采取以下措施：
- 警告并要求整改
- 限制或暂停部分服务
- 冻结账户余额
- 永久封禁账户
- 保留追究法律责任的权利

### 10. 协议终止

#### 10.1 用户终止

用户可随时通过注销账户终止本协议。

#### 10.2 平台终止

平台在以下情况下有权终止本协议：
- 用户违反本服务条款或隐私政策
- 用户从事欺诈、非法或滥用行为
- 根据法律法规要求
- 平台停止运营相关服务

#### 10.3 终止后果

协议终止后：
- 用户应停止使用平台服务
- 平台将按照退款政策处理用户余额
